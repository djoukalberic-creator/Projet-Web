# Projet-Web
notre projet de site web de propositios de voyage: the world of travelers
il sera divisé principalement (pour le moment) en 5 fichiers:
db.php pour la passerelle   avec MYSQL
connexion.php pour e login
pagePirncipale.php pour la page principale
design.php pour l'ensemble du design et du footer
listes_country.php pour la liste des pays
avez-vous des question?
pour être efficaces, je vous conseille d'installer Gtihub desktop, facile, visuel et nous pourrons éviter d'écraser nos fichiers, il y' "juste" quelques commandes à comprendre:
1. Le "Fetch Origin" (Vérifier le ciel)
C'est la première chose à faire en ouvrant le logiciel.

C'est quoi ? Tu demandes à GitHub : "Est-ce que mes camarades ont envoyé du code pendant que je dormais ?".

Pourquoi ? Si ton pote a modifié le header.php et que tu ne le sais pas, tu risques de travailler sur une vieille version.

2. Le "Pull" (Récupérer)
Si le Fetch a trouvé des changements, le bouton devient Pull origin.

Action : Tu télécharges les modifications des autres sur ton PC.

Note : Fais-le toujours AVANT de commencer à coder ta journée.

3. Le "Commit" (Enregistrer localement)
Une fois que tu as fini de moderniser ton bouton ou de réparer une balise dans pagePrincipale.php.

Action : Dans la colonne de gauche, tu coches tes fichiers, tu écris un petit message (ex: "Fix: balise <a> dans le tableau") et tu cliques sur Commit to main.

Attention : Ce n'est pas encore sur Internet ! C'est juste "sauvegardé" dans l'historique de ton PC.

4. Le "Push" (Propulser)
C'est l'étape finale.

Action : Tu cliques sur Push origin.

Résultat : Tes modifications sont envoyées sur le serveur GitHub. C'est seulement à ce moment-là que tes collègues peuvent voir ton travail.

5. La "Branch" (La zone de sécurité)
C'est le bouton tout en haut au milieu.

Le conseil de pro : Au lieu de travailler directement sur la branche main (la version officielle), crée une branche appelée test-boulangerie ou v2-ivory.

Avantage : Si tu fais une erreur fatale, le site officiel du groupe n'est pas touché. Tu ne fusionnes ta branche avec la main que quand tout fonctionne parfaitement.
